A full-stack ML-based app to detect vitamin deficiency detection from face images using ResNet152V2 with Flask backend and React frontend.

##Due to GitHub file size limits, the model file is not included. Please download it from Kaggle:

Model URL: https://www.kaggle.com/code/pangasainarendra/vitamin-deficiency-final-updated/output 
Dataset URL : https://www.kaggle.com/code/pangasainarendra/vitamin-deficiency-final-updated/input

After downloading, place the model file inside: backend/model/

Setup
Backend
pip install -r requirements.txt python app.py

Frontend
npm install npm run dev
